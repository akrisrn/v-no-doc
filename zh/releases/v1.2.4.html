v1.2.4

The documentation is still worked in process, and it maybe lacks a lot of content.

  1. 新增了 [noCommon] 标记。
  2. [redirect] 标记的优先级调整到了行内脚本之后。
  3. 嵌入文件的参数语法从 {{ }} 调整为 << >>（避免和 Vue.js 插值语法冲突）。
  4. 更新了一些 API：
    1. vno.mainSelf 的 getBacklinks 方法重命名为 loadBacklinks。
    2. vno.markdown 的 replaceInlineScript 方法重命名为 updateInlineScript。
    3. vno.utils.evalFunction 的返回值从字符串调整为数组。
    4. vno.markdown.renderMD 新增了默认为 true 的第二个参数。
    5. EMark 新增了 noCommon 枚举值。
  5. 新增了一些 API：
    1. vno.mainSelf.redirectTo。
    2. vno.articleSelf.redirectTo。
