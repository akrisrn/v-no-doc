asyncResults

v1.2.9 +

  • 类型：TAsyncResult[]

异步行内脚本会在执行完成后将它的结果 push 进这个列表。v-no 会监听它，在列表发生变化后取最后一项更新页面。

你也可以不经由异步行内脚本，直接用它手动控制需要异步加载的内容。

用法：

  • 将下面这行代码作为行内脚本，在页面上生成一个有 ID 的 Sync 元素占位：
return vno.element.getSyncSpan('async-script-999');
JavaScript
  • 将上一步的 ID 和结果字符串一起 push 进 asyncResults：
vno.articleSelf.asyncResults.push({ id: 'async-script-999', result: 'Hello World' });
JavaScript

这里为你准备了一个 Sync 元素，你可以在浏览器控制台中输入这行代码让它不要再转了 → 。
