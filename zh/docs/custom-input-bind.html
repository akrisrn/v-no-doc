自定义输入绑定

The documentation is still worked in process, and it maybe lacks a lot of content.

自定义输入绑定是一组响应特定按键输入的事件函数。

聚焦在页面时（输入框（input、textarea）除外）输入的按键会记录在 vno.appSelf.keyInput，v-no 会在它发生变化时遍历绑定列表匹配字符串结尾，执行第一个匹配到的函数。

你可以使用 vno.utils.addInputBinds 增加输入绑定，vno.utils.inputBinds 记录了所有绑定。

示例：

vno.utils.addInputBinds({
  hello: () => {
    alert('Greeting');
  }})
JavaScript

默认绑定：

  • home：返回主页。
  • dark：切换暗色模式。
  • zen：切换禅模式。
  • G：滚动到页面顶部。
  • gg：滚动到页面底部。
  • Backspace：按下退格键就好，移除最后一个输入字符。
