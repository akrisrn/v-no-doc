折叠标题

The documentation is still worked in process, and it maybe lacks a lot of content.

v-no 渲染的顶级标题（<article> 的直接子元素）除了会带有锚点图标（右侧），还有一个指示标题层级的元素（左侧），点击它就可以折叠归属于这个标题下的所有元素。

你可以在标题标记和文本之间添加一个加号（## + heading）来默认折叠指定标题。

示例：

  • heading 2-1
    • heading 3-1
      • heading 4-1
        • heading 5-1
          • heading 6-1
          • heading 6-2
        • heading 5-2
      • heading 4-2
    • heading 3-2
  • heading 2-2
    • heading 3-1
    • heading 3-2
