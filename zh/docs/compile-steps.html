最通常不过的 Node.js 项目编译：

  1. 首先确认你安装有 Node.js 和 Yarn 包管理器。
  2. 然后克隆或下载 v-no 项目仓库到本地。
  3. 进入项目目录，执行：
yarn && yarn build
