列表排序规则

The documentation is still worked in process, and it maybe lacks a lot of content.

对于搜索页和使用生成列表的页面中会出现的一类自动生成的列表项，v-no 有几条对它们进行排序的规则：

（v1.2.9 之后简化为最后两条）

  1. 有日期的在前。
  2. 首次更新日期较新的在前。
  3. 有标签的在前。
  4. 标签数较多的在前。
  5. 依次比较标签字符串，用第一组不相同标签的比较结果排序。
  6. 用标题字符串的比较结果排序。
  7. 用路径字符串的比较结果排序。

它们之间是层层递进的关系，如果前一条规则无法确定前后，会继续用下一条判断。
