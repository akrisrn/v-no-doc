目前存在的一些问题

The documentation is still worked in process, and it maybe lacks a lot of content.

这里列出的问题在 v-no 的预期表现之外，并且还没能 / 无法解决。

  1. 如果当前页面有图片正在加载，浏览器有可能会阻止 Hash 路由进行跳转（地址变了但是页面没有变）。这时你要么等待图片加载完成，要么就只能刷新页面。
  2. 因为 Hash 路由地址占用了页面锚点的功能，我只能在它上面重新实现一个锚点。新锚点使用标题层级加序号进行索引，还有一堆不尽如人意之处。
  3. 嵌套 details 语法会出现开闭匹配错误。
