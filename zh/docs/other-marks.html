其它语法标记

The documentation is still worked in process, and it maybe lacks a lot of content.

  • 不引入通用文件
  • 只在搜索页生效的标记
    • 搜索框
    • 搜索结果列表
    • 查询到的结果条数
    • 总查询条数
    • 查询用时（秒）

不引入通用文件

[noCommon]，需要单独占据一行。

只在搜索页生效的标记

搜索框

[input]，需要单独占据一行，参数为输入框占位字符串。

搜索结果列表

[result]，需要单独占据一行，参数为查询开始前展示的字符串。

查询到的结果条数

[number]，参数为默认值。

总查询条数

[count]，参数为默认值。

查询用时（秒）

[time]，参数为默认值。
